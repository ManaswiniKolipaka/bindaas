Write queries using BSON sytax.

All queries must follow following syntax { "_operation" : operation , "_operation_args" : operation_args } where operation can be either of find,count,group or mapreduce

Depending on the operation , operation_args must be specified as follows

{ "query" : { ... } , // BSON object "fields" :{ ... }, "skip" : skip count - integer (optional), "limit" : limit count - integer (optional) }
count : Count all objects matching the query { "query" : "query" : { ... } , // BSON object }
group : Group operation { "key" : { ... } , // BSON object "condition" : { ... } , "initial" : { ... } , "reduce" : ... , "finalize" : ... }
mapreduce : MapReduce operation { "query" : { ... } , // BSON object "map" : ... , "reduce" : ... , "outputCollection" : ... }
